# 🎟️ Ingressos App

Projeto desenvolvido para praticar lógica de programação, manipulação do DOM e interação com elementos HTML utilizando JavaScript.

---

## 📌 Sobre o projeto

O **Ingressos App** é uma aplicação simples que simula a compra de ingressos para diferentes setores, permitindo ao usuário selecionar a quantidade desejada e verificar a disponibilidade em tempo real.

Durante o desenvolvimento, foram aplicados conceitos fundamentais de programação, como validações, controle de estado e atualização dinâmica da interface.

Este projeto também faz parte do meu portfólio, demonstrando a criação de aplicações interativas utilizando JavaScript puro.

---

## 🚀 Funcionalidades

- Selecionar tipo de ingresso
- Informar quantidade desejada
- Validar disponibilidade de ingressos
- Atualizar quantidade restante em tempo real
- Exibir mensagens de erro e sucesso
- Interação dinâmica com a interface

---

## 🧠 Conceitos praticados

- Variáveis e manipulação de dados
